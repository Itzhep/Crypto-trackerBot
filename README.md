# pytrogram Bot 
this bot have admin panel and you can with 
```bash
/admin
```
command open admin tools like :
```bash
block an member
send an message to all the users in bot database
/stats command
/block
/unblock
```
# to install 
```bash
pip install pyrogram==2.0.106
pip install pycoingecko==3.1.0
```
